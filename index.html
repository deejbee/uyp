Up Your Pass (UYP) automates the tedious and time consuming logging in process of banks and financial web sites, for FREE! It goes much further than any other password manager by handling multi-page login processes that require specific letters from passwords, and more!.

UYP itself is available here as a Google Chrome extension https://chrome.google.com/webstore/detail/up-your-pass/kblhemffnhfhjianafepcclpakocicgg

Presets for a range of banks and other financial organisations are available and is being added to all the time (you can help too!)

Features:

  • Automated login in seconds!
  • Growing catalog of financial organisations
  • Add your own
  • Edit login strategies
  • Share your strategy (not your passwords)
  • Graph numbers over time
  • Investment value tracking
  • Encrypted using Web Cryptography API
  • AES-GCM 256 level encryption
  • Sensitive data NOT on external servers

Contributors

If you have defined a login strategy for an organisation not already listed and you think others would benefit then please create a pull request in GitHub. You can use the Export function in the UYP Chrome Extension. This will produce the JSON output (no passwords are included) to add to the 'access_definitions.json' file.
